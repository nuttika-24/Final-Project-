User Stories

1.ในฐานะผู้ใช้ ฉันต้องการกดปุ่มตัวเลข เพื่อให้สามารถใส่ค่าตัวเลขลงในเครื่องคิดเลขได้

2.ในฐานะผู้ใช้ ฉันต้องการกดปุ่มเครื่องหมายบวก เพื่อให้สามารถบวกค่าระหว่างตัวเลขสองตัวได้

3.ในฐานะผู้ใช้ ฉันต้องการกดปุ่มเครื่องหมายลบ เพื่อให้สามารถลบค่าระหว่างตัวเลขสองตัวได้

4.ในฐานะผู้ใช้ ฉันต้องการกดปุ่มเครื่องหมายคูณ เพื่อให้สามารถคูณค่าตัวเลขได้

5.ในฐานะผู้ใช้ ฉันต้องการกดปุ่มเครื่องหมายหาร เพื่อให้สามารถหารค่าตัวเลขได้

6.ในฐานะผู้ใช้ ฉันต้องการกดปุ่มเท่ากับ (=) เพื่อให้ระบบแสดงผลลัพธ์ของการคำนวณ

7.ในฐานะผู้ใช้ ฉันต้องการให้ระบบป้องกันไม่ให้หารด้วยศูนย์ เพื่อให้การใช้งานปลอดภัยและไม่มี error

8.ในฐานะผู้ใช้ ฉันต้องการปุ่ม Clear (C) เพื่อให้สามารถล้างค่าทั้งหมดบนหน้าจอได้ง่าย

9.ในฐานะผู้ใช้ ฉันต้องการให้ระบบแสดงข้อความ Error เมื่อฉันใส่ข้อมูลผิด เพื่อให้รู้ว่าต้องแก้ยังไง

10.ในฐานะผู้ใช้ ฉันต้องการให้ระบบแสดงตัวเลขบนหน้าจอแบบเรียลไทม์ เพื่อให้ดูอินพุตที่กำลังพิมพ์อยู่ได้

11.ในฐานะผู้ใช้ ฉันต้องการให้ระบบป้องกันการกดเครื่องหมายซ้ำ เพื่อให้ผลลัพธ์ไม่ผิดพลาด

12.ในฐานะผู้ใช้ ฉันต้องการ UI แบบปุ่มกดง่ายๆ เพื่อให้ใช้งานได้สะดวกแม้ไม่เคยใช้มาก่อน
